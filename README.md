## Readbit Learning System

#### Setup

- Requires Python3.9

1. `git clone https://github.com/unluckyvevo/IT2201-Project/`
1. `cd IT2201-Project/`
1. `python3 -m pip install -r requirements.txt `
1. `python3 run.py`
1. Go to `http://127.0.0.1:5000/` to view the web application
